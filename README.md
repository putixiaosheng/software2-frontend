## 项目运行

项目运行环境为微信开发者工具版本号（2.19.4）

* 项目运行前请确保写在node.js

* 导入vantapp组件库指令

```
npm i @vant/weapp -S --production
```

* 构建npm模块

## 功能简介

本项目设计的是一个基于微信小程序的计算器界面，在实现普通计算器和科学计算器功能的基础上，提供利息计算和利率修改的功能。项目请求的发送全部基于本地实现。
